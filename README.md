<h2 id="technologies">💻 Technologies</h2>

- ASP.NET Core
- C#
- EF Core
- JWT
- .NET

<h2 id="started">🚀 Getting started</h2>

<h3>Prerequisites</h3>

- .NET
- Git

<h3>Cloning</h3>

```bash
git clone https://github.com/gaabrieltorres7/dotnet-auth
```

<h3>Starting</h3>

```bash
cd dotnet-auth
dotnet run
```
